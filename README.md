# barocert.sdk.php7
---
barocert SDK for PHP7   

KakaoCert
1. 본인인증 API
2. 전자서명(단건) API
3. 전자서명(복수) API
4. 출금동의 API
5. 간편로그인 API

NaverCert
1. 본인인증 API
2. 전자서명(단건) API
3. 전자서명(복수) API
4. 출금동의 API

PassCert
1. 본인인증 API
2. 전자서명 API
3. 출금동의 API
4. 간편로그인 API

### Requirements

php 7.1.3+   
php curl module   
openssl 1.0.1g+ (don't forget about heartbleed.)
